# Agenda — Smart Productivity & Task Management App

Agenda is a minimalist and modern productivity application designed to help users organize tasks, manage schedules, track routines, and stay focused throughout the day.

Built using AI-assisted development tools including Antigravity, Codex, and ChatGPT, the application combines clean UI/UX principles with lightweight performance and responsive design.

---

## Overview

Agenda is more than just a traditional to-do list app. It integrates multiple productivity-focused utilities into a single streamlined interface, helping users manage both daily work and personal routines efficiently.

The platform focuses on:
- Task management
- Appointment organization
- Calendar planning
- Sleep tracking
- Work & record management
- Clean distraction-free productivity experience

---

## Features

### Task Management
- Add, complete, and delete tasks
- Active and completed task filters
- Real-time task statistics
- Clear completed tasks functionality

### Productivity Utilities
- Appointment tracking
- Calendar integration UI
- Sleep monitoring section
- Work/record management module

### User Experience
- Minimal modern interface
